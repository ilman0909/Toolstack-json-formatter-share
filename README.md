# Toolstack JSON Formatter

🚀 JSON Formatter online sederhana dan cepat.  
💡 Format, minify, validasi, dan bagikan JSON hanya dengan satu klik.

## ✨ Fitur Utama
- ✅ Format JSON otomatis
- 🧹 Minify JSON
- 📋 Copy to Clipboard
- 📥 Download hasil JSON
- 🔗 Shareable Link
- 🌙 Dark Mode (akan datang)

## 🛠️ Teknologi
- React + Vite
- Tailwind CSS (optional)
- Hosted on Vercel
